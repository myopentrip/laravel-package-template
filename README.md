## About Laravel Package
This is template for creating package on laravel.

### Installation
```php
compose require :vendor_slug/:package_slug
```

### Usage
```php
$package = new Myopentrip\LaravelPackage();
echo $package->sayhello();
```

### Testing
```
compose test
```
